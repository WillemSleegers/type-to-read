# Type to Read

A typing practice app that lets you improve your typing speed and accuracy while reading content you want to read.

## Features

- Real-time character-by-character feedback (correct/incorrect highlighting)
- Scrolling 3-line window that follows your typing
- WPM, accuracy, and error tracking statistics
- Load custom text via paste, file upload, or built-in samples
- Customizable settings (font size, punctuation, capitalization)
- Dark/light mode

## Tech Stack

Next.js 16, React 19, Tailwind CSS 4, Radix UI, TypeScript
